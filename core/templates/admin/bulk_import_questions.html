Simplified CSV Format:

  • question_text: The question text (required)

  • answer_1_correct to answer_4_correct: true/false for correct answers

Note:

  • Difficulty will be set to "medium" by default
  • Points will be set to 1 by default
  • Explanation and hint will be empty by default
